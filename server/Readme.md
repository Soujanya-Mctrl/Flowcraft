# Server dmaid.cloud

## Technologies

- REST API
- MongoDB
- Express
- Node.js
- TypeScript
- Google Gemini

entry point: `src/index.ts`

## How to Run

1. Open the root dir.
2. Move to server:

```bash
cd server
```

3. Install dependencies:

```bash
npm install
```

4. Run the server:

```bash
npm run dev
```
